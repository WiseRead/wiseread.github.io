About

WiseRead was built for the community to enjoy.
Just drag/select your comic/manga/manhwa and read.

Also, you can enter direct download url of your zip/cbz file.
No server, no trackers.


Multi-Chapters mode

You can select more than one zip/cbz file and enter Multi-Chapters mode.
In this mode there are arrows to navigate between the chapters.

By default the arrows direction is left-to-right, but you can
set it to right-to-left with the 'ltr=false' param:
https://wiseread.github.io/?ltr=false


Online Reader

You can create Online Reader link by combining the download url
with the reader url.

Like this:
https://wiseread.github.io/?download=https://example.com/files/chapter.cbz

And set params:
* separator=on | separator=off (turn off when it's webtoon)
* theme=light | theme=dark

Like this:
https://wiseread.github.io/?separator=on&theme=dark&download=https://example.com/files/chapter.cbz

NOTE:
The 'download' param must be the last param.
